核心特性

为效率而生

Qtine 将复杂性封装在框架内部，让你专注于业务逻辑。

插件热加载

拖入 .zip 即用，修改即生效——无需重启。每个插件独立沙箱运行，互不干扰。

零重启开发体验

写代码、保存、拖入 WebUI，插件即刻上线。热加载机制让你告别反复重启，迭代速度提升 10 倍。

< 1s热加载耗时

三阶段消息管道

预处理 → 处理 → 后处理，每一层都是可插拔的钩子。拦截黑名单、过滤关键词、注入上下文——所有操作都有明确的切入点。

WebUI 管理后台

可视化仪表盘、插件一键启停、消息日志追踪、全局设置管理。不用敲命令，所有操作在浏览器完成。

9个管理页面

插件市场集成

浏览、搜索、一键安装社区插件。支持多镜像源加速，自定义源配置，本地插件管理。

WebUI 管理后台

仪表盘、插件管理、消息日志、设置中心，一站式操作。深色主题，现代化设计。

事件总线

发布/订阅模式的事件系统，松耦合架构让插件协作更优雅。

双协议 WebSocket

同时支持反向 WS 与正向 WS，兼容 OneBot V11，适配任意 Bot 后端。

插件市场

一键安装社区插件，镜像源自动加速，支持官方源与自定义源。

快速开始

一行命令跑起来

Python 3.8+ 即可运行，从安装到第一条消息只需 3 分钟。

Terminal
# 安装 Qtine
pip install qtine

# 初始化配置
qtine init

# 启动框架
python main.py

# 打开管理后台
# → http://localhost:4990/webui
