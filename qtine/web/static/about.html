项目简介

Qtine 是一个基于 Python 的模块化聊天机器人框架，原生支持 OneBot V11 协议，可对接 NapCat、Lagrange 等主流 QQ 协议端。

采用插件 + 适配器架构，内置 WebUI 控制台、事件总线、消息管道、存储后端与权限系统，开箱即用，亦可深度定制。
